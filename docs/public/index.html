    Introduction #

    A lightweight excel app that helps you schedule basic school (or anything with cohorts of people) activities. Uses a configurable list of rules to let you know if the schedule if accurate.

    Using Rules #

    On the Rules sheet you can setup all the various restrictions you want to apply to your schedule.

    That way as your updating the schedule it will warn you if you have broken one of your rules.
